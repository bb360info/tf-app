# /dev — Перезапуск dev-сервера

Используй когда сервер завис, не отвечает или порт 3000 занят чужим процессом.

## Шаги

// turbo
1. Убить всё что висит на порту 3000:
```bash
kill $(lsof -ti :3000) 2>/dev/null; sleep 1; echo "Port 3000 cleared"
```

// turbo
2. Запустить dev-сервер:
```bash
pnpm dev 2>&1
```

3. Проверить что сервер ответил HTTP 200:
```bash
curl -sL --max-time 20 -o /dev/null -w "HTTP %{http_code}" http://localhost:3000/ru 2>&1
```

Если вернул **HTTP 200** — готово ✅  
Если **000 или 404** — проверь вывод `pnpm dev` на ошибки компиляции TypeScript.
