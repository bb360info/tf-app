## Tech Debt Scanner Workflow

Используй для оценки состояния проекта и поиска кандидатов на рефакторинг.

1. Подсчёт использования `any` в TypeScript файлах. Тебе нужно использовать инструмент `grep_search` с `Query: " any"` или `Query: ": any"`. Обрати внимание, сколько совпадений вернет `grep_search`.

2. Поиск `TODO`, `FIXME` и `HACK` комментариев в коде. Воспользуйся `grep_search` по папке `src/`.

3. Составь отчёт `Tech Debt Score` на основе результатов:
- Количество `any` типов (цель: 0).
- Открытые `TODO`/`FIXME` (сгруппированные по критичности).
- Топ-3 кандидата на рефакторинг (по твоему мнению).

4. Если пользователь попросит, начни процесс устранения долга через воркфлоу рефакторинга.
